## 二、单题 JSON 字段说明

```javascript
{
  // ── 基础字段 ──
  id: "indef_001",                 // 唯一ID
  sentence: "Cuando _____ ...",     // 带挖空的例句
  translation: "当我到家时...",      // 中文翻译
  verb: "llegar",                  // 原形动词
  person: "yo",                    // 目标人称
  tense: "indefinido",             // 目标时态
  answer: "llegué",                // 正确答案（含重音）
  answer_no_accent: "llegue",      // 无重音版本（用于黄灯判断）
  rule: "cuando + 时间点 → Indefinido", // 口诀
  distractor: "llegaba",           // 典型干扰项
  distractor_error: "llegaba 是...", // 干扰项解释
  difficulty: "A2",                // 难度级别

  // ── 功能四字段（干扰项反馈）──
  error_type: "orthographic",      // 错误类型英文标识
  error_type_cn: "正字法变化遗漏",    // 错误类型中文
  distractor_2: "llegó",           // 第二干扰项
  distractor_2_error: "...",       // 第二干扰项解释
  distractor_2_type: "person_confusion",
  similar_verbs: [                 // 同类动词（最多3个）
    { verb: "pagar", yo_form: "pagué" }
  ],

  // ── 功能六字段（规律卡片）──
  rule_tag: "orthographic_gar"     // 关联到 rule_cards.js 中的规律
}
```

### error_type 可选值

| error_type | 中文名称 | 用于什么场景 |
|-----------|---------|------------|
| `tense_confusion` | 时态混淆 | indefinido vs imperfecto |
| `orthographic` | 正字法变化遗漏 | -gar/-car/-zar 动词 |
| `stem_change` | 词干变音遗漏 | e→ie, o→ue |
| `person_confusion` | 人称混用 | yo/él 形式搞混 |
| `accent_missing` | 重音缺失 | llegue→llegué |
| `wrong_paradigm` | 套用错误变位规则 | 不规则动词按规则变 |

### rule_tag 可选值（对应 rule_cards.js）

```
orthographic_gar, orthographic_car, orthographic_zar,
stem_eie_pres, stem_oue_pres, yo_irregular_pres,
fully_irregular_indef, accent_indef_yo_el,
imperfect_regular, subj_present_regular,
reflexive_pres, ser_ir_same_indef
```

## 三、大模型生成题库 Prompt

### Prompt 1：批量生成混淆专项题目

复制以下 Prompt 发送给 DeepSeek / ChatGPT / Claude：

```
你是一名西语教学专家。请为西语学习 App 生成练习题，严格按照以下 JSON 格式输出。
要求：
1. 生成 10 道「Indefinido vs Imperfecto」混淆专项题目
2. 难度为 A2-B1，面向中国西语学习者
3. 例句必须自然、贴近生活
4. 每道题必须包含以下所有字段：

{
  "id": "indef_XXX",
  "sentence": "带有 _____ (原形动词) 的西语例句",
  "translation": "中文翻译",
  "verb": "原形动词",
  "person": "yo/tú/él/nosotros/vosotros/ellos",
  "tense": "indefinido 或 imperfecto",
  "answer": "正确变位（含重音符号）",
  "answer_no_accent": "去掉重音的版本",
  "rule": "一句话中文口诀，说明为什么用这个时态",
  "distractor": "最常见的错误答案",
  "distractor_error": "解释为什么这个答案是错的，≤40字",
  "difficulty": "A2 或 B1",
  "error_type": "从 tense_confusion/orthographic/stem_change/person_confusion/accent_missing/wrong_paradigm 中选一个",
  "error_type_cn": "对应中文名称",
  "distractor_2": "第二种常见错误",
  "distractor_2_error": "解释",
  "distractor_2_type": "错误类型",
  "similar_verbs": [{"verb": "同类动词","yo_form": "变位形式"}],
  "rule_tag": "从以下值选：orthographic_gar/orthographic_car/orthographic_zar/stem_eie_pres/stem_oue_pres/yo_irregular_pres/fully_irregular_indef/accent_indef_yo_el/imperfect_regular/subj_present_regular/reflexive_pres/ser_ir_same_indef，如不适用填 null"
}

请直接输出 JSON 数组，不要加其他解释文字。
```

### Prompt 2：批量生成变位表数据

```
你是一名西语语法专家。请为以下动词生成完整变位表 JSON 数据，时态为「简单过去时 (indefinido)」。

动词列表：estar, poder, saber, querer, decir, venir, poner, traer, dar, ver

格式要求：
{
  "verb": "动词原形",
  "tense": "indefinido",
  "tense_cn": "简单过去时",
  "is_irregular": true/false,
  "difficulty": "A2/B1/B2",
  "rule_tag": "对应规律标签或null",
  "conjugation_table": {
    "yo": { "answer": "变位形式", "answer_no_accent": "去重音版本" },
    "tu": { ... },
    "el": { ... },
    "nosotros": { ... },
    "vosotros": { ... },
    "ellos": { ... }
  },
  "common_mistake_persons": ["容易出错的人称"]
}

请直接输出 JSON 数组。
```

## 四、补充题库后的操作流程

```
1. 复制大模型输出的 JSON
2. 粘贴到对应数据文件中（注意数组格式和逗号）
3. 运行 npm run build 确认无语法错误
4. git add . && git commit -m "feat: 扩充题库" && git push
5. Cloudflare Pages 自动部署
```

> **建议**：每次用大模型生成后，人工抽检 5-10 道题的答案正确性，特别是不规则动词的变位。
